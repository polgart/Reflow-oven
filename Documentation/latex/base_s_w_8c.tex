The communication state machine is implemented in \mbox{\hyperlink{base_s_w_8c_af341203893341c4fc0aca6617827cc78}{Receive\+FTDI\+\_\+callback()}} function. This function is always when new data is arrived from the FTDI UART-\/\+USB bridge. The current state is the function of the prevoiuos state and one or more incoming bytes.
